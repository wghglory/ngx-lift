A spinner is a visual indicator of an ongoing, user-initiated process.

Clarity has two types of spinners, each with its own use case:

  • Page Spinners: For tracking the progress of an operation related to an entire page.
  • Inline Spinners: For tracking the progress of an operation related to a specific component.

The spinner has below inputs:

  • size: Determines the spinner size. Accepts 'lg' (default), 'md', or 'sm'.
  • center: Centers the spinner as a block element (default). Pass false to disable centering.
  • inline: Specifies whether the spinner should be displayed as an inline-block element. Useful for aligning spinners within text or other inline elements.
  • class: A string of additional CSS classes to apply to the spinner.
